# TypeScript 学习指南

## 简介
TypeScript 是一种由 Microsoft 开发和维护的开源编程语言，它是 JavaScript 的一个超集，添加了可选的静态类型和对 ES6+ 的支持。

## 安装
你可以通过 npm 安装 TypeScript：
```bash
npm install -g typescript
```

## 基础语法
- 变量声明
- 函数
- 类
- 接口

## 高级主题
- 泛型
- 装饰器
- 异步编程
